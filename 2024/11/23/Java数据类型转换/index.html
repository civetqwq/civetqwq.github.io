Java 类型转换

类型转换是将一个原始数据类型的值赋给另一个类型的过程。

在 Java 中，有两种类型转换：

  1. 自动类型转换（Widening Casting）

    • 自动转换：将较小的数据类型转换为较大的数据类型。
    • 转换顺序：byte -> short -> char -> int -> long -> float -> double

    例如，byte 可以自动转换为 int，int 可以自动转换为 long，等等。这种转换是自动进行的，无需显式地指定。

  2. 手动类型转换（Narrowing Casting）

    • 手动转换：将较大的数据类型转换为较小的数据类型。
    • 转换顺序：double -> float -> long -> int -> char -> short -> byte

    这种转换需要手动强制转换，因为可能会丢失数据或精度。为了进行这种转换，通常需要使用强制类型转换的语法，如 (targetType) value。
