前言:

Java的基础语法：

对象：对象是类的一个实例

类：是一个模板，它描述一类对象的行为和状态。

方法：是行为，一个类可以有很多方法。逻辑运算、数据修改以及所有动作都是在方法中完成的。

实质变量：每个对象都有独特的实例变量，对象的状态由这些实例变量的值决定。

一. 第一个Java程序 它将输出字符串 Hello World

可能遇到的问题：

  1. 大小写：每个单词的大小写不能出现问题，Java是大小写敏感的
  2. 类：对于所有的类来说，类名的首字母应该大写。如果类名由若干单词组成，那么每个单词的首字母应该大写
  3. 方法：所有的方法名都应该以小写字母开头。如果方法名含有若干单词，则后面的每个单词首字母大写
  4. 源文件名：源文件名必须和类名相同，保存文件时因该注意（大小写敏感）
  5. 主方法入口：所有的 Java 程序由 public static void main(String[] args) 方法开始执行
  6. 符号： 用英文的，如果用中文符号也会报错

二. Java程序运行机制

  • 编译型 complie：把我们写的代码转换成计算机可以执行的代码（类似于直接把一本汉语书翻译成英文版）

    执行速度快，对操作系统的要求低，操作系统一般会用，c/c++ ……语言都是编译行的

  • 解释型：(我说一句你解释一句，回头去要一个东西，又要重新给我翻译)

​ 比如: 网页——>对于速度要求没那么快可以用到解释型语言（边执行边解释）

  • 程序运行机制 ：将一系类的指令按照有规律的操作进行执行

    源程序( @.java文件) —-> Java编译器 —-> 字节码(@.class) —-> 类装载器 —> 字节码校验器 —> 解释器 —-> 操作系统平台

三. Java标识符

Java 所有的组成部分都需要名字。类名、变量名以及方法名都被称为标识符

注意事项：

  • 所有的标识符都应该以字母（A-Z 或者 a-z）,美元符（$）、或者下划线（_）开始
  • 首字符之后可以是字母（A-Z 或者 a-z）,美元符（$）、下划线（_）或数字的任何字符组合
  • 关键字不能用作标识符,(关键字指具有特殊意义和功能的预定义词汇,如：if，int，public等)
  • 标识符是大小写敏感的
  • 合法标识符举例：age、$salary、_value、__1_value
  • 非法标识符举例：123abc、-salary

小总结：标识符大小写敏感且不能以数字和 - 开头，开头之后可以是任何字符组成，关键字不能用作标识符
