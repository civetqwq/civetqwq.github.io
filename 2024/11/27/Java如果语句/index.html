概述：

Java 条件和 if 语句

Java 支持数学中常见的逻辑条件：

  • 小于：a < b
  • 小于或等于：a <= b
  • 大于：a > b
  • 大于或等于：a >= b
  • 等于：a == b
  • 不等于：a != b

可以使用这些条件来执行不同的操作，根据不同的决策进行选择。

Java 有以下条件语句：

  • if：用于指定在指定条件为真时执行的代码块。
  • else：用于指定在条件为假时执行的代码块。
  • else if：用于指定当第一个条件为假时需要测试的新条件。
  • switch：用于指定多个替代的代码块执行。
