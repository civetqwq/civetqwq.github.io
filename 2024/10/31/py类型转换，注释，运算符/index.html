一.数据类型转换：

A.隐式类型转换

  1. 在隐式类型转换中，Python 会自动将一种数据类型转换为另一种数据类型;我们对两种不同类型的数据进行运算，较低数据类型（整数）就会转换为较高数据类型（浮点数）以避免数据丢失

从输出中可以看出，整型和字符串类型运算结果会报错，输出 TypeError。 Python 在这种情况下无法使用隐式转换。

但是，Python 为这些类型的情况提供了一种解决方案，称为显式转换.

B.显式类型转换

  1. 在显式类型转换中，用户将对象的数据类型转换为所需的数据类型。 我们使用 int()、float()、str() 等预定义函数来执行显式类型转换

二. python注释

分为 # ， ‘ ’ ， “ ” ，三种。

#：单行注释

在 Python 中，多行注释是由三个单引号 ‘’’ 或三个双引号 “”” 来定义的，而且这种注释方式并不能嵌套使用。

当你开始一个多行注释块时，Python 会一直将后续的行都当作注释，直到遇到另一组三个单引号或三个双引号。

3 .python赋值运算

就是个一个变量设值，进行运算。

海象运算符： 它使用 := 符号。这个运算符的主要目的是在表达式中同时进行赋值和返回赋值的值。

使用海象运算符可以在一些情况下简化代码，在需要在表达式中使用赋值结果的情况下。这对于简化循环条件或表达式中的重复计算很有用。

海象：

  1. if (n := 10) > 5: ：这是使用海象运算符（:=）的写法。海象运算符在表达式中进行赋值操作。
  • (n := 10)：将变量 n 赋值为 10，同时返回这个赋值结果。
  • > 5：检查赋值后的 n 是否大于 5。如果条件为真，则执行接下来的代码块。
  1. print(n)：如果条件为真，打印变量 n 的值（即 10）。
  • 传统写法需要单独一行来赋值 n，然后在 if 语句中进行条件检查。而使用海象运算符的写法可以在 if 语句中直接进行赋值和条件检查。
