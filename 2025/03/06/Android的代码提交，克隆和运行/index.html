一. 提交github 代码

步骤：

进入cmd 中你要提交的文件里（例如：cmd F:\daimacang>）接着逐步进行一下操作

git

git add . 暂存区, 将当前目录下的所有修改过的文件添加到 Git 的暂存区，准备进行提交

git commit -m init 进行一次 Git 提交，提交消息是 "init"， inti 可以更改

git push 用于将本地仓库的提交上传到远程仓库，从而同步代码并与其他协作者共享更改

完成

二. 克隆

git clone 命令用于从远程仓库复制一个完整的代码库到本地计算机。这个命令不仅复制了项目的文件，还包括了该项目的历史版本记录

步骤：

在GitHub中找到自己所需文件，复制 http…. , 在cmd中进入想要存放文件的盘，在cmd中运用 git clone https: // ….. ，

三. 真机运行 Android

使用真机，

首要步骤：开启开发者模式，打开 USB 接口，打开 停用adb授权超时功能，打开 验证可调试的字节码

其次连接进行连接

一. 网络连接：

  1. 网络连接，真机和pc端需使用同一个网络，在pc端中的Android里右侧的Divece manager 中连接好设备，运行即可，运行时会自动下载且运行所写

二. usb连接：

  1. 运用usb接口(数据线连接)，在pc端中的Android里右侧的Divece manager 中连接好设备，运行即可，运行时会自动下载且运行所写, 速度快
