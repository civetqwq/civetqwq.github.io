for-each 循环遍历 HashMap 的项

只需要 键，使用 keySet() 方法；只需要 值，使用 values() 方法

a. 示例：keySet()和values()

values()

b. Print keys and values

解：

  • 程序使用 HashMap 来存储描述颜色的键值对："天空" 和 "蓝色"，"小草" 和 "绿色"。

  • 使用 for-each 循环遍历 HashMap 中的所有键，并打印每个键和它对应的值。

  • HashMap 是一个存储键值对的数据结构，可以通过键来访问值。

  • keySet() 方法返回所有键，get() 方法用于根据键查找对应的值。
